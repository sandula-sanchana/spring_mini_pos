# Mini POS System (AJAX + Spring Boot)

A simple **Mini POS (Point of Sale) System** built using **Spring Boot (REST API)** and a **frontend with AJAX** for smooth, reload-free CRUD operations.  
This project covers common POS functions like managing customers/items and handling orders.
